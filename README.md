# dotfiles
My .file config

for vim file
```bash
ln -s ~/dotfiles/vim ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
```

fot tmux
```bash
ln -s ~/dotfiles/tmux ~/.tmux
ln -s ~/.tmux/tmux.conf ~/.tmux.conf
```
